# taro-vue3-template
taro vue3 rapid development template

## support

- html tag
- unocss
- auto-import（不支持H5平台）
- vue-components-unplugin (插件目前热更不生效，需要手动重启，不支持H5平台)

## install

```
yarn
```

### weapp

```
yarn dev:weapp
```

## unocss

关于unocss：

尽量不要使用attribute的方式去编写样式，经测试，小程序不能完全兼容。

[unocss官网](https://uno.antfu.me/)
